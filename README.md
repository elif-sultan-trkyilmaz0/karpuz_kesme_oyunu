#  Karpuz Kesme Oyunu

## Proje Hakkında

Bu proje, C++ ve Qt Framework kullanılarak geliştirilmiş bir karpuz kesme oyunudur. Oyuncunun amacı, belirli süre içerisinde ekranda rastgele konumlarda beliren karpuzları tıklayarak mümkün olduğunca fazla skor elde etmektir.

## Oyun Özellikleri

* Rastgele konumlarda karpuz oluşturma
* Karpuzların animasyonlu hareket etmesi
* Karpuz kesme mekaniği
* Süre tabanlı oyun sistemi
* Kesilen ve kaçırılan karpuz sayısını takip etme
* Skor kaydetme
* Maksimum skor (High Score) sistemi

## Kullanılan Teknolojiler

* C++
* Qt Framework
* Qt Widgets
* QTimer
* QPropertyAnimation
* QFile ve QTextStream

## Oyun Kuralları

* Oyuncunun başlangıç süresi 30 saniyedir.
* Karpuzlara tıklanarak kesilir.
* Kesilen karpuzlar skora eklenir.
* Kaçırılan karpuzlar ayrı olarak sayılır.
* Oyun sonunda skor kaydedilir ve en yüksek skor ile karşılaştırılır.
